설정

Post·Log·Diary 데이터는 이 브라우저에만 저장돼요. 파일로 백업해두면 다른 기기에서도 이어볼 수 있어요.

📤 데이터 백업하기

Post 편지함, Log 방·편지, Diary 감정 기록을 하나의 파일로 내보내요. (사진은 용량 문제로 포함되지 않아요 — 아끼는 사진은 내 사진첩에서 직접 다운로드해주세요.)

📥 백업 파일 가져오기

다른 기기에서 받은 백업 파일을 불러와요. 기존 기록은 지워지지 않고, 겹치지 않는 내용만 추가돼요.

🔔 자동 백업 리마인드

브라우저 데이터는 언제든 지워질 수 있어요. 주기적으로 백업하도록 알려드릴까요?
