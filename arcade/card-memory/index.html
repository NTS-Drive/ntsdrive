How to Play

Objective

20 cards, 10 pairs. Half the cards show an emoji, the other half show its English word — flip two cards to match an emoji with its word.

Roles

Intern / Manager / Director only change your starting lives (5 / 3 / 2). A wrong match costs one life; run out and the run ends.

Each stage draws its 10 pairs at random from a themed pool of ~40 items (20 for Crazy Mode), so you'll see a different set nearly every run — memorizing exact positions from a past run won't help much.

Preview

Every stage briefly shows all cards face-up before flipping them down — but the window gets shorter each stage (see table above), down to no preview at all by Stage 5.

Time limit

Once you flip a first card, a timer starts for that stage's limit (10s down to 5s by Stage 5). If you don't flip a second card before it runs out, the first card flips back automatically and it counts as a miss — same as a wrong match.

Bonus cards

Each stage secretly marks 1–3 cards as bonus cards. Matching a pair that includes one restores +1 life (up to your role's starting max) with a celebration banner — matching one while already at full lives just gives you the heads-up, no extra life to gain.

Hints

You get exactly one use each of two hints per run (not per stage):

  • Reveal All — shows every unmatched card face-up again for 4 seconds.
  • Flash Pair — highlights one still-hidden pair's border without flipping it.

Both are only usable between turns (not while a first card is already flipped and the timer is running).

Scoring

Each correct match is worth 10 points. Two or more correct matches in a row multiply that match's points: 1 + 0.1 × (streak − 1), capped at 2.0× — a wrong match resets the streak. Clearing a stage with zero wrong matches earns a +20 perfect-clear bonus.

Save, Resume & Reset

  • Progress auto-saves after every match attempt.
  • Reopening this page resumes your run automatically, mid-stage.
  • Reset wipes your current run and lets you pick a new role. Top 10 records are kept.

Records

The Top 10 is ranked by score. Reaching the all-time #1 automatically downloads a screenshot of your run.
